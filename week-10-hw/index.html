Responsive Web Design

What is it?

In today's world, there are so many different ways to view web content. You can log on to Facebook on your laptop, your phone, your iPad or even your Kindle. All of these devices vary in screen size, and that is where responsive web design comes in. Responsive web design allows content on a website to look good on any device.

What we do with it?

Responsive web design uses HTML and CSS elements to not leave out any info on a site. Such elements include:

  • Viewport: sets the visible area of a web page
  • Media Query: limit the style sheets' scope by using media features, such as width, height, and color
  • Flex Box: creates boxes that rearrange themselves on a page as its dimensions change.

The Process

It should also be aethetically pleasing, which is where the plannning comes in. It is important to base your style elements around your content, so always plan your content first.
